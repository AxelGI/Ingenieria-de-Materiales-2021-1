¡Bienvenido !

El paso del tiempo conlleva cambios y a su vez, los cambios traen consigo  el surgimiento de nuevas necesidades, para satisfacer estas se requiere innovar y diseñ​ar materiales que se adapten y en el mejor de los casos faciliten las tareas para las que son necesarios. 

Los diagramas de fase son fundamentales para estos fines, pues comprender el comportamiento de los materiales en ciertas condiciones como alta temperatura, agentes corrosivos, impactos, etc. es fundamental para saber si es funcional para desempeñar el trabajo para el que son creados, además apoyan, entre otros, estudios de solidificación, microestructura y metalurgia con el fin de facilitar el proceso de fabricación y aplicación.

¡Es por eso que diseñamos este sitio web para que puedas comprender mejor estos diagramas de fase!
